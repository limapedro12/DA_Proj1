Detailed Description

A class that represents a railway segment between two Stations, being an Edge of the network graph.

Given that we represent the railway network as a graph, the edges/segments are the graph's edges.
An Edge is represented by its origin and destination Stations (note that the Edges are bidirectional) and its capacity in terms of the number of trains that can travel on it simultaneously (weight).
Note: all the methods time complexity is O(1), except where specified.
