Detailed Description

A class that represents a Station, being a node of the network graph.

Given that we represent the railway network as a graph, the stations are the graph's nodes.
A Station is represented by its name, pointers to its District and its Municipality, its Township (Parish - Freguesia) and the line it belongs to (North Line, Minho Line...). Contains its adjacent railway segments.
Note: all the methods time complexity is O(1), except where specified.
